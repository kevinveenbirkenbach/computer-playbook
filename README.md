# Client Playbook
[![License: AGPL v3](https://img.shields.io/badge/License-AGPL%20v3-blue.svg)](https://www.gnu.org/licenses/agpl-3.0)

Playbook to setup Manjaro GNOME clients.
# Setup

Run:
```bash
ansible-galaxy collection install -r requirements.yml
```
